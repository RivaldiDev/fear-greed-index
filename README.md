# Crypto Sentiment - Fear & Greed Index

Bitcoin market sentiment tracker with Fear & Greed Index visualization and 7-day historical data

## Features

- Gauge visualization for current sentiment score
- 7-day historical trend with color-coded cards
- Sentiment zones: Extreme Fear to Extreme Greed
- Multiple indicator breakdowns with animated bars
- Emoji-based sentiment display
- Amber/blue gradient dark theme

## Tech Stack

- **Framework:** Next.js 16 (App Router, TypeScript)
- **UI:** shadcn/ui + Tailwind CSS
- **Animations:** Framer Motion
- **API:** alternative.me Fear & Greed API (Free)
- **Deployment:** Vercel
